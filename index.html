FAQs

How does it work?

The Japanese Name Converter uses a combination of dictionary lookup, substitution rules, and machine learning to convert English characters into katakana.

For common English names, a dictionary lookup of about 1,300 English names is used. For other names, a learned substitution model trained on these 1,300 names is applied instead. This method is very similar to the Transformation-Based Learner (TBL) invented by Eric Brill.

Essentially, given a list of English/Japanese name pairs, the system learns a series of substitution rules to apply to the English input in order to get the Japanese output. For instance, the first rule the system learns is to replace the letter "L" with the letter "R", because there is no L/R distinction in Japanese. Later on. more subtle rules are applied, such as "replace G with J if it's followed by an E." Here is the full list of rules.

Hey, doofus, you messed up my name! My name's Daenerys Targaryen, and your system got the last vowel wrong!

Congratulations, you took high-school Japanese. This web site was not designed for you.

The machine learning method sometimes makes mistakes. In my own tests, it had an accuracy of about 95% on a per-character basic, but your mileage may vary. In any case, always check with a Japanese friend before getting an ill-advised tattoo based on this web site. (And check with your Mom too. Mom always knows best.)

In my defense, I can say that transliteration is not easy, especially when you're dealing with a language as orthographically mangled as English. The vowel system is very irregular, and furthermore, many names are ambiguous. For instance, is Jaime pronounced /'jeɪmi:/ ("JAY-mee") or /'haɪmeɪ/ ("HIGH-may")?

In any case, don't expect perfect results, and you won't be disappointed.
